# Model Setup

## 1. Download Qwen-VL from  [website](https://huggingface.co/Qwen/Qwen-VL/tree/main).

## 2. Download YOLOv10 from  [website](https://github.com/THU-MIG/yolov10).

## 3. Prepare Model


The directory will be as follows.
```
VLMIdentification
├── Qwen-VL
│   ├── config.json
│   ├── generation_config.json
|   |...
│   ├── pytorch_model-00001-of-00010.bin
|   |...
│   ├── pytorch_model-00010-of-00010.bin
|   |...
├── yolov10x.pt.
|...
```
